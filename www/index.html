Kurulum

Dakikalar içinde çalışır hale gel

1

Sunucuya bağlan

Ubuntu 20.04 / 22.04 / 24.04 veya Debian 11 / 12 çalıştıran bir sunucuya root erişimiyle bağlan.

2

Kurulum scriptini çalıştır

bash <(curl -fsSL https://raw.githubusercontent.com/mwlih28/mc-manage-panel/main/scripts/install-panel.sh)
3

Paneli aç, kullanmaya başla

Domain adresinizi tarayıcıda açın. SSL otomatik kurulur, admin hesabınız hazır olur.

📋 Gereksinimler: Ubuntu/Debian, min. 1GB RAM, 10GB disk, root erişimi
